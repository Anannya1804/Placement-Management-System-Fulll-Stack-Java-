# 📖 <font size="6">Overview</font>

<font size="4">

The **Placement Management System** is a powerful and user-friendly Java desktop application designed to simplify and automate placement-related activities for students and administrators.

The system efficiently manages:
- Student Records  
- Company Details  
- Placement Activities  
- Recruitment Information  

through an interactive graphical user interface built using **Java Swing** and **AWT**.

</font>

---

# ✨ <font size="6">Features</font>

<font size="4">

✔️ Student Registration & Management  
✔️ Placement Status Tracking  
✔️ Company Information Management  
✔️ Interactive GUI using Java Swing & AWT  
✔️ Secure Data Storage using `.dat` Files

# 📌 <font size="6">Project Objective</font>

<font size="4">

The primary objective of this project is to digitize and simplify placement-related activities by reducing manual paperwork and improving efficiency in managing placement records and recruitment processes.

This system helps students and administrators maintain organized placement information with better accessibility and usability.

</font>
